CSS

En aquest lloc web explicarem l'ús de CSS en HTML5.



Fulls d'Estil a Cascada (de l'anglès C ascading S tyle S heets ) o CSS és el llenguatge d' estils utilitzat per descriure la presentació de documents HTML o XML (en-US)(incloent diversos llenguatges basats en XML com SVG , MathML o XHTML ). CSS descriu com ha de ser renderitzat l'element estructurat a la pantalla, en paper, a la parla o en altres mitjans.

CSS és un dels llenguatges base de l' Open Web i té una especificació estandarditzada per part del W3C. Anteriorment, el desenvolupament de diverses parts de les especificacions de CSS era realitzat de manera sincrònica, la qual cosa permetia el versionat de les recomanacions. Probablement haureu escoltat sobre CSS1, CSS2.1, CSS3. Tot i això, CSS4 mai s'ha llançat com una versió oficial.

Des de CSS3, l'abast de les especificacions es va incrementar de forma significativa i el progrés dels diferents mòduls de CSS va començar a mostrar diverses diferències, fet que va fer més efectiu desenvolupar i publicar recomanacions separades per mòduls . En comptes de versionar les especificacions de CSS, la W3C actualment realitza una captura de les últimes espcificacions estables de CSS .
